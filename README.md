# Activity Log

日常の文化的活動（コンサート、映画鑑賞、読書など）を記録・管理する Flutter アプリケーション。

## 技術スタック

- Flutter 3.x
- Firebase Realtime Database
- HTTP REST API
- fl_chart（統計グラフ）
- flutter_dotenv（環境変数管理）

## アーキテクチャ

- サービスクラスによるデータ層の分離
- 再利用可能なウィジェットコンポーネント
- 環境変数による設定管理

## 機能

- 活動の記録（6 種類のカテゴリー）

  - コンサート 🎵
  - 美術館・博物館 🏛
  - 読書 📚
  - 旅行 ✈️
  - 映画 🎬
  - 音楽 🎼

- 活動評価（6 種類）
- 活動の編集・削除
- カテゴリー別フィルタリング
- 統計情報の可視化（グラフ表示）
